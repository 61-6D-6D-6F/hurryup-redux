# HurryUp - redux

HurryUp is a moblie phone application for locating, registering and reviewing restrooms around the world.

This repository contains the state management part of the [React Native](https://reactnative.dev/) frontend.

## Technologies:
[Redux.js](https://redux.js.org/) - state management

[redux-presist](https://www.npmjs.com/package/redux-persist) - persist and rehydrate a redux store

[redux-persist-sensitive-storage](https://www.npmjs.com/package/redux-persist-sensitive-storage) - storage engine to use react-native-sensitive-info with redux-persist
